# HiAn Cake — Flipbook

Catalog flipbook 3D cho **HiAn Cake** (HTML/CSS/JS thuần).

## Chạy local

Mở `index.html` trong trình duyệt, hoặc dùng Live Server / server tĩnh:

```bash
npx --yes serve .
```

## Cấu trúc

- `index.html` — layout sách
- `style.css` — giao diện (Hoa Đào)
- `script.js` — lật trang, ảnh (`BOOK_IMAGES`), mobile
- `anh/` — ảnh sản phẩm (`1.jpg` … `87.jpg`, `logo.jpg`)
